To create a saperate environment for any machine learning project you have to first isolate all
the dependencies to a perticular project


command to creaet a conda environment
 $ conda create -p venv python==version -y

 this command will intiallize a saperate environment for your project


after this to activate this environment you have to run this command
 $conda activate C:\Users\hp\Desktop\ml_project_structure\venv


 to deactivate this environment run this
 conda deactivate
